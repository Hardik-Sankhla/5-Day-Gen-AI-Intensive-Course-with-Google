# 5-Day Gen AI Intensive Course with Google

Curated resources, guides, and codelabs from the "5-Day Gen AI Intensive Course with Google" (originally live: March 31–April 4, 2025). This repository collects notes, example notebooks, and walkthroughs to help learners follow the course material at their own pace.

Author: Hardik Sankhla

## What’s included
- Day-by-day guides (Day 1 → Day 5) covering LLM foundations, embeddings, agents, domain-specific LLMs, and MLOps for GenAI.
- Codelab examples and Kaggle Notebook links.
- Notes, sample prompts, evaluation scripts, and optional advanced resources.

## Setup (local)
1. Clone the repo:

   git clone https://github.com/Hardik-Sankhla/5-Day-Gen-AI-Intensive-Course-with-Google.git
2. Create a Python virtual environment and install dependencies (if provided):

   python -m venv .venv
   .\.venv\Scripts\activate
   pip install -r requirements.txt
